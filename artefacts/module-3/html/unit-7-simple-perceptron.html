def sum_func(inputs, weights):
    return inputs.dot(weights)

This function calculates the dot product of the inputs and weights arrays using the .dot() method of NumPy arrays. The dot product is a mathematical operation that is commonly used in linear algebra and is particularly useful in the context of neural networks for performing the weighted sum of inputs
