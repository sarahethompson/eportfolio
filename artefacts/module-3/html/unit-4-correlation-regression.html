Removing Missing Values from both datasets¶

When calculating a correlation between two variables, the calculation typically involves multiplying or summing values from both variables for each observation. If an observation has a missing value in either of the variables we are trying to correlate, that observation cannot be included in the calculation.

Most statistical software and libraries (like pandas in Python) will automatically exclude rows with missing values when computing correlations. However, explicitly removing missing values beforehand gives you more control and transparency over which data is being used. It also helps avoid potential errors or unexpected results that can occur when working with functions that might not handle missing values gracefully.

Essentially, to get an accurate correlation coefficient, we need complete pairs of data for the variables we are analyzing. Removing missing values ensures that our correlation calculation is based only on these complete pairs.
