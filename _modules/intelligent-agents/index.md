## Overview

This module covered the theory and practice of autonomous software agents, covering architectures, BDI reasoning, multi-agent coordination, and communication languages, culminating in a team design project and individual implementation.

I came to it with a software engineering background rather than an AI research one, which made the formal agent models feel familiar in structure but genuinely new in framing. The learning outcomes, including analysing agent architectures, applying agent techniques to real-world problems under uncertainty, building a working system, and contributing as part of a distributed team, mapped closely onto work I was already doing professionally on a Risk Informed Deployment Platform at HashiCorp, which gave me a useful anchor for evaluating the theory.
