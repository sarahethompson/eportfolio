# Introduction to Intelligent Agents

## Overview

Agent-based computing is an approach to software design in which autonomous entities called **agents** interact with each other and their environment in order to achieve goals. This paradigm has become increasingly important as computing environments have become more distributed, complex and dynamic.

Traditional centralised systems can struggle to manage environments where many components must operate independently. Agent-based systems address this challenge by distributing decision-making across autonomous agents that can respond to changes in their environment.

## Trends Driving the Rise of Agent-Based Systems

### Distributed Computing

Modern systems operate across distributed infrastructures such as:

- cloud platforms
- microservices
- Internet of Things (IoT) networks
- large-scale data systems

In these environments it is often impractical for a single system component to control all decisions.

### Increasing System Complexity

Many modern systems involve large numbers of interacting components. Agent-based systems provide a way to manage this complexity by assigning responsibilities to specialised agents.

### Automation and Autonomy

Organisations increasingly require systems that can operate with minimal human intervention. Agents are designed to act autonomously and make local decisions based on their environment.

### Advances in Artificial Intelligence

Recent advances in artificial intelligence, particularly machine learning and large language models, have renewed interest in autonomous software agents capable of reasoning and decision-making.

## Definition of an Agent

An agent is a system that:

- perceives its environment
- acts upon that environment
- operates autonomously to achieve goals

Russell and Norvig define an intelligent agent as a system that perceives its environment and takes actions to maximise the likelihood of achieving its goals.

## Characteristics of Agents

According to Wooldridge, intelligent agents typically possess several key properties.

### Autonomy
Agents operate independently without direct human control.

### Reactivity
Agents respond to changes in their environment.

### Proactiveness
Agents can take initiative in order to achieve goals.

### Social Ability
Agents can interact and cooperate with other agents.
