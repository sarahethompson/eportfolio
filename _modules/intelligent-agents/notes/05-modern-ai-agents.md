# Modern AI Agents

## Symbolic AI vs Machine Learning

Artificial intelligence historically developed along two main approaches.

### Symbolic AI

Symbolic AI represents knowledge using logical rules and symbols.

Examples include:

- expert systems
- rule-based systems
- BDI agents

### Machine Learning

Machine learning systems learn patterns from data rather than relying on explicit rules.

Examples include:

- neural networks
- supervised learning
- deep learning

## Large Language Models

Large language models (LLMs) are neural network models trained on large datasets.

They are often used in modern agents for:

- reasoning
- planning
- code generation
- information retrieval

## Multi-Agent Systems with AI Agents

Modern multi-agent systems may consist of multiple specialised agents that collaborate to perform tasks.

Example architecture:

- Coordinator agent
- Bug analysis agent
- Security analysis agent
- Complexity analysis agent

Each agent performs a specialised function, and the system combines their outputs to support decision-making.

## Emergent Behaviour in Multi-Agent Systems

When multiple agents interact, complex system behaviour can emerge from their interactions.

For example, specialised analysis agents in a DevOps pipeline could collectively determine whether software changes should be deployed.

No single agent performs the entire decision-making process, but their combined interactions produce system-level intelligence.
