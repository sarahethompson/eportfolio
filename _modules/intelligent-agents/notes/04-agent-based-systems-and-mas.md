# Agent-Based Systems and Multi-Agent Systems

## Single-Agent Systems

A single-agent system consists of one autonomous agent operating in an environment.

Examples include:

- robotic vacuum cleaners
- automated trading bots
- monitoring systems

## Multi-Agent Systems (MAS)

A multi-agent system (MAS) consists of multiple agents interacting with each other to solve problems.

Agents may:

- cooperate
- coordinate
- negotiate
- compete

Multi-agent systems are particularly useful when tasks require distributed expertise or coordination between multiple components.

## Agent-Based Modelling (ABM)

Agent-Based Modelling is used to simulate complex systems composed of interacting agents.

Unlike multi-agent systems, which are designed to perform real tasks, ABM focuses on understanding system behaviour.

Researchers use ABM to study how global system behaviour emerges from local agent interactions.

Common applications include:

- modelling financial markets
- studying organisational behaviour
- analysing social systems
- modelling disease spread

## Emergent Behaviour

Emergent behaviour occurs when complex system behaviour arises from the interactions of many simple agents.

Examples include:

- ant colony behaviour
- traffic flow patterns
- market dynamics

Agent-based modelling is often used to study these emergent phenomena.

## ABM vs MAS

| Concept | Purpose |
|---|---|
| ABM | Simulation and analysis of complex systems |
| MAS | Building real systems with multiple interacting agents |
