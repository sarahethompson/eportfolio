# Unit 1 Study Notes – Introduction to Intelligent Agents

## 1. Core Idea
- Intelligent agents perceive their environment and act to achieve goals.
- Agent behaviour is typically modelled as a mapping from perceptions to actions.

## 2. Agent Properties
- **Autonomy**: operates without direct human intervention.
- **Reactivity**: responds to changes in the environment.
- **Proactiveness**: takes initiative toward objectives.
- **Social ability**: can communicate with other agents or systems.

## 3. Common Agent Types
- Simple reflex agents.
- Model-based reflex agents.
- Goal-based agents.
- Utility-based agents.
- Learning agents.

## 4. Reflection Prompt
- Which agent architecture best fits a real-world problem you care about?
- What trade-offs exist between simplicity and performance?
