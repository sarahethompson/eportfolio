## Summary
This unit introduces intelligent agents, why agent-based systems are useful in modern distributed computing, and the core properties that define agent behavior.

## Key topics
- Agent-based computing in distributed and complex systems
- Core agent properties: autonomy, reactivity, proactiveness, social ability
- First-order logic as a foundation for symbolic reasoning
- Agent architecture families: reactive, deliberative, and BDI
- Multi-agent systems, coordination, and emergent behavior
